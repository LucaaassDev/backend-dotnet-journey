# Backend .NET Journey

## Objetivo
Este repositório documenta minha jornada de aprendizado para me tornar um Desenvolvedor Backend utilizando C# e .NET, com foco em entrar no mercado de trabalho (estágio ou vaga júnior).

O objetivo não é apenas aprender sintaxe, mas desenvolver a capacidade de construir APIs, entender regras de negócio, trabalhar com banco de dados e escrever código organizado e versionado como um profissional.

---

## Stack
- Linguagem: C#
- Plataforma: .NET (Console / ASP.NET Core)
- Versionamento: Git e GitHub
- Banco de dados (em breve): SQL Server ou PostgreSQL

---

## Estrutura atual
- Projeto Console inicial para prática dos fundamentos da linguagem
- Exercícios focados em lógica, tipos, métodos e controle de fluxo

---

## Próximos passos
- Consolidar fundamentos de C#
- Iniciar estudos em ASP.NET Core
- Criar APIs REST
- Integrar banco de dados com Entity Framework Core
- Desenvolver um projeto backend completo
- Preparar portfólio para aplicação em vagas

## Observação
Este repositório representa aprendizado contínuo. O código pode evoluir, ser refatorado e melhorado conforme o avanço dos estudos.
